Definition of Terms

A LINEAR FUNCTION is defined by f(x) = mx + b, where m and b are real numbers.
Its graph is a line with slope (m) and y-intercept(b).

slope (m) y-intercept (b) degree of function line
≠ first diagonal
0 ≠ 0 horizontal
0 0 not defined no line

Remember:

While all linear equations produce straight lines when graphed,
NOT all linear equations produce linear functions.
In order to be be a linear function, a graph must be:
both linear (a straight line) and
a function (matching each x-value to only one y-value).
