Definition of Terms

The set of all first coordinates in ordered pairs is called the DOMAIN of the relation.
The set of all second coordinates is called the RANGE of the relation.

A correspondence maybe classified as one-to-one, many-to-one, or one-to-many.

See the examples below:

One-to-One

It is one-to-one correspondence if every element in the domain is mapped
to a unique element in the range.
Example:
Domain: {1,2,3,4}
Range: {5,6,7,8}

Many-to-One

It is many-to-one correspondence if any 2 or more elements in the domain
are mapped to the same element in the range.
Example:
Domain: {1,2,3,4}
Range: {5,6}

One-to-Many

It is one-to-many correspondence if one of the element in the domain
is mapped to 2 or more elements in the range.
Example:
Domain: {1,2}
Range: {3,4,5,6}
